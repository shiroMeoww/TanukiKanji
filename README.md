# TanukiKanji
stack technologiczny:
React + TypeScript: pozwala tworzyć nowoczesny i przejrzysty frontend
Node.js z Express zapewnia prosty i szybki backend
MongoDB dobrze sprawdza się przy danych o zmiennej strukturze, takich jak użytkownicy, lekcje i postępy nauki. Będzie to strona do nauki kanji
